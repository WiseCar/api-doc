## 2. 获取行程详情
### URL
`GET /api/mobile/trip/{tripId}`

### 请求参数

| 参数名称 | 说明 | 举例 |
| -------- | ---- | ---- |
| tripId | URL占位符参数, 行程ID | 904 |
